# ID2222 Data Mining - Homeworks Repository

This repository contains all the code developed for the **ID2222 Data Mining** course at KTH Royal Institute of Technology. The course covers fundamental techniques and scalable algorithms for processing and mining large datasets, including frequent item discovery, similarity detection, data stream processing, graph partitioning, and spectral clustering.

The repository includes five homework assignments, each tackling a key concept in data mining using various algorithms and tools. All tasks have been implemented in Python.
